# **PlagiAI**
### *AI-Powered Multi-Document Detection of AI-Generated Content and Semantic Plagiarism*

---

## üìå Project Overview
PlagiAI is an AI-based system designed to detect **AI-generated text** and **semantic plagiarism** across multiple documents.  
Unlike traditional plagiarism checkers that only match exact words, this system understands **meaning and context** using **Transformer-based NLP models**.

It generates an **Authenticity Score** and provides a **color-coded report** that distinguishes:
- AI-written text
- Paraphrased / semantically plagiarized text
- Human-written / original text

---

## üéØ Key Objectives
- Detect **AI-generated** content using **BERT**-based classification.
- Identify **semantic plagiarism** using **Sentence-BERT + FAISS** similarity search.
- Support **multi-document cross comparison**.
- Produce **explainable and downloadable authenticity reports**.

---

## üß† Key Features

| Feature | Description |
|--------|-------------|
| AI Content Detection | Detects text generated by models like ChatGPT, GPT-4, Gemini, etc. |
| Semantic Plagiarism Detection | Finds rephrased or meaning-level copied text. |
| Multi-Document Support | Compares content across multiple uploaded documents. |
| Color-Coded Highlighting | Shows AI, plagiarized, and original text clearly. |
| Easy Web Interface | Upload, analyze, and download reports easily. |
